debug options


'/rate:: newrate newstop' change refresh rate in sec
'/stop' stop refresh
'/info' current user info
'/change:: newname_newcolor' change current user(string) color(css)
'/del:: number' delete custom amount messages on server
'/msg:: json' custom message:
'/msg:: {"userMessage":"hello","date":"2022-11-03T12:26:47.589Z","userId":101010,"userName":"custom","userColor":"red"}'
'/saveHistory' back up history (wont be deleted after clearCache)
'/baseUrl:: url' set new base url
'/apiKey:: key' set new api key (optional)
